# 数字经济仿真实验 3：公共物品博弈与算法治理

## 📖 实验简介
本实验旨在通过 Python 仿真模拟数字经济中的“公共资源池”场景（如开源社区、UGC平台），探究“搭便车”行为对集体利益的损害，并对比分析“惩罚”与“奖励”两种算法治理机制的效果。

## 📂 文件结构
*   `streamlit_app.py`: 实验主程序（Web 交互界面）。
*   `simulation_exp3.py`: 独立的仿真脚本（用于批量生成数据）。
*   `data_exp3_public_goods.csv`: (示例) 标准数据结构参考。
*   `requirements.txt`: 项目依赖库。

## 🚀 如何运行
1.  **安装依赖**：
    ```bash
    pip install -r requirements.txt
    ```
2.  **启动实验平台**：
    ```bash
    streamlit run streamlit_app.py
    ```
3.  **访问网页**：
    在浏览器中打开终端显示的 URL（通常为 http://localhost:8501）。

## 📊 实验步骤
1.  **阅读大纲**：在首页了解实验背景与参数定义。
2.  **运行仿真**：进入“仿真控制台”，设置玩家数量 $N$ 与增值系数 $M$，点击“开始仿真”。
3.  **下载数据**：仿真完成后，下载生成的 CSV 文件用于后续分析。
4.  **分析图表**：在“数据分析”页面观察贡献率曲线与收益分布。
5.  **提交报告**：结合数据回答思考题。

## 🧠 核心算法逻辑
*   **收益公式**：$\pi_i = (E - c_i) + \frac{\sum c_j \times M}{N}$
*   **惩罚逻辑**：若 $c_i < 0.8 \times \bar{c}$，则 $\pi_i = 0$。
*   **奖励逻辑**：若 $c_i > \bar{c}$，则 $\pi_i = \pi_i + 5$。
